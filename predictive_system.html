Smart Predictive System

A full-stack mobile application that integrates an Android frontend with a Python Flask API to deliver real-time machine learning predictions using Linear Regression, Decision tree and ANN (Compare the charges) directly to the user.

View GitHub Repo Live Demo / APK

The Problem & Solution

The Problem

Users lacked a way to get accurate, on-the-go predictions without relying on heavy client-side processing. Local machine learning models were draining battery life and proving difficult to update seamlessly on mobile devices.

The Solution

I developed a lightweight Android frontend that outsources heavy computation. By establishing a global Python Flask API, the mobile app sends data securely to a cloud environment where trained Regression and Classification models process the data and return instant results.

Technical Deep Dive

Instead of packing models into the APK, I deployed a Flask server. This allowed me to leverage Python's robust data science ecosystem. The server accepts JSON payloads, processes the data through trained models, and returns the predictions.

@app.route('/predict', methods=['POST'])
def predict():
    data = request.get_json()
    prediction = model.predict(data['features'])
    return jsonify({'result': prediction[0]})

To ensure a personalized user experience, I integrated Firebase for secure user registration and login. Furthermore, all search histories and prediction results are pushed to the Firebase Realtime Database, allowing users to view their past activity seamlessly within the Android RecyclerView.

To reduce manual data entry friction, I integrated OCR capabilities. Users can snap a picture of standard forms, and the app extracts the relevant text, formats it, and auto-populates the input fields before sending the data to the Flask API.
